# 🍔 Hahn Burger — Template HTML/CSS

Projeto simples de **treinamento e prática de HTML e CSS puro**, simulando a interface de uma hamburgueria fictícia chamada **Hahn Burger**.
O objetivo é praticar **estruturação de layout, responsividade e organização de estilos** sem o uso de frameworks.

---

## 📷 Preview

Interface de uma landing page contendo:

* Header com navegação
* Hero section
* Campo de busca de produtos
* Grid de produtos
* Footer

Layout totalmente construído com **HTML5 e CSS3**.

---

## 🚀 Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **Google Fonts**

  * Inter
  * Calistoga

## 📱 Responsividade

O layout foi adaptado para diferentes tamanhos de tela:

| Dispositivo | Comportamento                   |
| ----------- | ------------------------------- |
| Desktop     | Grid com 3 colunas              |
| Tablet      | Grid com 2 colunas              |
| Mobile      | Grid com 1 coluna e menu mobile |

Breakpoints utilizados:

```
820px
500px
```
